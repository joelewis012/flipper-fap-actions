Connect Your GitHub

FlipperFAP uses GitHub Actions to compile your .fap files for free. You need a GitHub Personal Access Token to trigger compilations.

1Fork the flipper-fap-actions repo to your GitHub account
2Go to github.com/settings/tokens → Generate new token (classic)
3Enable scopes: repo and workflow → click Generate
4Paste the token below. It's saved only in your browser (never sent to any server).
